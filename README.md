# Soham Neswankar — Personal Portfolio

A modern personal portfolio website built with **HTML, CSS, and JavaScript**, designed to showcase my projects, skills, and experience through a distinctive neo-brutalist user interface.

🌐 **Live Demo:** https://soham5128.github.io/My-Portfolio/

---

## Overview

This portfolio serves as a central place to present my work, technical skills, and development journey. The project focuses on creating a memorable user experience through custom animations, interactive elements, and a unique visual design while maintaining fast performance and responsiveness.

Unlike traditional portfolio templates, this site was built from scratch using vanilla web technologies without frameworks or build tools.

---

## Features

* Responsive design for desktop, tablet, and mobile devices
* Neo-brutalist visual style with bold typography and strong visual hierarchy
* Interactive animations and scroll effects
* Project showcase section
* Skills and technology overview
* Interactive timeline and journey section
* Terminal-inspired resume experience
* Fast-loading static website with no framework dependencies

---

## Design Approach

The portfolio is inspired by **neo-brutalism**, a design style characterized by:

* Bold layouts and typography
* High-contrast color palettes
* Thick borders and visible structure
* Flat shadows and geometric elements
* Creative, handcrafted interactions

The goal is to create a portfolio that stands out while remaining easy to navigate and visually engaging.

## Running Locally

Clone the repository:

```bash
git clone https://github.com/SOHAM5128/My-Portfolio.git
```

Navigate to the project directory:

```bash
cd My-Portfolio
```

Start a local development server:

```bash
npx serve .
```

or

```bash
python -m http.server 8000
```

Open your browser and visit:

```text
http://localhost:8000
```
